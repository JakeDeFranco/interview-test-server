Autonomous Car Assignment

Build an application / tool in React or Python with the following criteria:
  • Fetches a car route from this API server
  • Determines if the route was a success or a failure
  • If the route results in a failure, determine the cause as well as the step and position where the failure occurred

There are different levels to this assignment, level 1 being the easiest, level 2 having some gotchas, etc.
If you complete a level with ease, feel free to jump to the next one. Please timebox your solution to 1 day max.

Level 1

  • Fetch a static autonomous car route from this server at: localhost:5000/autonomous-car/routes/

Level 2

  • Fetch a random autonomous car route: localhost:5000/autonomous-car/routes/random/
Warning: this endpoint has gotchas
